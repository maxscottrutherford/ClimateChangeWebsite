About

My name is Max Rutherford and I created this project for a summer POLS1 class at Sacramento State. I'm also a computer science major, so when I heard a website was an option for our end-of-semester voter guide project, I got started right away. As I've used this project to practice web design, I've also become more involved in the issue of climate change. I truly do believe it is a crisis. I hope that with this website, I've provided a resource for people to become more educated on the topic and also show people what they can do to help stop it from becoming worse.
